Detailed Description

Inventory item base class for items that have physics properties ( rigid body, collider etc... ). By adding ITEM_COLLIDE_WITH_ENVIROMENT define enable PhysicsInventoryItem to use item collision with enviroment with additional cost of more physics objects on scene. To use - assign colliders to item 'impactColliders' field. Not using default colliders because those surround all item and can penetrate enviroment outside just weapon impact zone - blade ( collision with ground etc..).
