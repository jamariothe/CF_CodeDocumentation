Detailed Description

This class is used to spawn canvas and SCENECONTROLLER object. If you dont want to create canvas and SCENECONTROLLER objets on every scene, you can use this class to spawn them and DontDestroyOnLoad - them That way they'll be saved to use across all scenes. Just place object with this coponent on scene and fill fields. Be sure to place canvas and SCENECONTROLLER prefabs ready in resources folder.
